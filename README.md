* To make easy && fast network scanning process.

#### Features
1. Host discovery
2. Identify OS 
3. TCP Port scan
4. Identifying service version

#### Downloading && Usage
```
git clone https://github.com/malbana77/BEHOLDA
cd BEHOLDA
chmod +x Beholda.sh requirements.sh
./requirements.sh
sudo ./Beholda.sh
```
